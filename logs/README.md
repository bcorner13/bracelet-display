# Logs

Project execution and validation logs:

- `parametric_audit.log` - Output from parametric system validation
- `print_validation.log` - 3D print test results and measurements
- `execution.log` - MCP tool execution logs and errors
- `recompute.log` - FreeCAD recomputation results

**Purpose:** Track project execution, validation results, and debugging information per PROJECT_BOOTSTRAP.md standard.
