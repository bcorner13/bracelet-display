# STL Files

Final exported STL files ready for 3D printing:

- `bracelet_display_v1.stl` - Production-ready model
- `bracelet_display_test.stl` - Test prints and validation

All STLs should be:
- Manifold (watertight) geometry
- Centered at origin (0,0,0) for K2 Plus
- Optimized for support-free printing
